#### 02. 유지보수 

만약 출력 내용을 "김수겸"은 5번, 신성민은 3번으로 변경해야 하는 경우 반복문을 활용한 코드에서는 각각 수정해줘야 하지만, 함수의 매개변수를 이용하면 유지보수 하기 좋은 멋진 코드를 만들 수 있습니다.

#### 함수를 사용하는 이유?
함수를 활용하면 중복 코드 제거 및 코드를 재사용할 수 있을뿐 아니라 유지보수하기 좋은 코드를 만들수 있습니다.

### 03_함수 구조 3가지

#### 함수와 관련 된 내용

> 메개변수
> 지역변수와 전역변수
> 리턴값
> 함수 호출
> 함수 정의
> 함수 리터럴
> 함수 이름 규칙

#### 첫번쨰 : 일반적인 함수
<pre><code>function 함수이름 () {
  실행구문;
}</code></pre>

#### 두번쨰 : 매개변수가 있는 함수
<pre><code>function 함수이름 ([매개변수1 [, 매개변수2 [,...]]) {
  실행구문;
}</code></pre>

#### 세번쨰 : 리턴값이 있는 함수
<pre><code>function 함수이름 ([매개변수1 [, 매개변수2 [,...]]) {
  실행구문;
  return 실행결과;
}</code></pre>
