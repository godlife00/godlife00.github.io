#### jquery에서 제공하는 스타일 관련 기능 중 핵심 내용만을 선별해 다룹니다. 스타일 관련해서 실무를 진행하기 위해 반드시 할 줄 알아야 하는 핵심 내용은 다음과 같습니다.

> 핵심 01 : 스타일 값 구하기

> 핵심 02 : 스타일 값 설정하기

> 핵심 03 : 클래스 추가

> 핵심 04 : 클래스 삭제

### 핵심 01_스타일 값 구하기

<pre><code>$대상.css("스타일 속성이름");

$대상.css(["스타일 속성이름" , "스타일 속성이름" ... ]);
</code></pre>

css() 메서드를 이용하면 글자 크기와 같은 스타일 속성값을 구할수 있습니다.

#### 예제1

[li항목의 글자 크기를 출력해주세요](https://jsfiddle.net/7196f4L8/1/)

- 글자크기속성을 css()메서드를 넣어 호출

#### 예제2

[메뉴의 너비와 높이를 출력](https://jsfiddle.net/dhbfaLvs/)

- 스타일 속성을 동시에 여러개 구할때는 스타일 속성이름을 배열 값으로 만들어 css()메서드를 호출

<br><br>
### 핵심 02_스타일 값 설정하기

<pre><code>$대상.css("속성이름", "값");

$대상.css({
  속성이름 : 값,
  속성이름 : 값
});
</code></pre>

css() 메서드는 스타일 속성값을 설정할때도 사용됩니다. 첫번째 매개변수는 속성이름, 두번짜 매개변수에는 속성값을 넣어 호출

#### 예제3

[실행버튼을 누를때마다 선택메뉴 아이팀의 글자 크기 증가](https://jsfiddle.net/jbuvw8h6/2/)

- 현재 폰트 크기를 css() 메서드를 이용해 구한 후 값에는 단위가 붙어있는 문자열이기 때문에 parseInt() 함수로 숫자로 형변환 후, 구한 값에 1을 더 한 값으로 폰트 크기를 설정해 줍니다.

#### 예제4

[실행버튼을 누르면 폰트크기를 11부터 시작해서 5씩 증가](https://jsfiddle.net/nd1ym0qe/6/)

- each() 메서드를 활용해 메뉴 갯수만큼 루프를 돌립니다.
- 루프 내부에서 css()메서드를 호출해 글자 크기를 설정
- 루프가 한번씩 돌때마다 글자 크기값을 5만큼 증가

#### 예제5

[실행버튼을 누르면 너비와 높이를 100px로 만들어주세요](https://jsfiddle.net/Lmqrvtu8/3/)

### 핵심 03_클래스 추가

<pre><code>$대상.addclass("클래스이름1 [클래스이름2 ...]");</code></pre>

addclass() 메서드를 이용하면 노드에 하나 이상의 클래스를 추가할수있습니다.

#### 예제6

[실행후 클릭한 메뉴 항목에 select 선택자 추가](https://jsfiddle.net/sbrcw1p7/4/)

- 여러개의 스타일 속성값을 적용하는 경우 css()메서드보다 addclass() 메서드를 사용하면 더 효과적입니다.

#### 예제7

[클릭한 메뉴 항목에 select와 border 선택자 추가](https://jsfiddle.net/p3mbfjw8/)

- addclass() 메서드를 사용하면 여러개의 스타일 선택자를 동시에 추가할수있습니다.
